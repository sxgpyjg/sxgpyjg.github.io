✅ Termux 永久备份 & 换机恢复全流程（复制版）

🟢 第一部分：旧手机（只需做一次）
# ================================
# 1. 进入博客仓库
# ================================
```bash
cd ~/gitdemo
```
# ================================
# 2. 创建配置备份目录
# ================================
```bash
mkdir -p config/termux
```
# ================================
# 3. 备份核心配置文件
# ================================
```bash
cp ~/.myfuncs.sh config/termux/
cp ~/.bashrc config/termux/
cp ~/.zshrc config/termux/
cp ~/.profile config/termux/
```
# ================================
# 4. 创建一键恢复脚本
# ================================
cat > config/termux/restore.sh <<'EOF'
#!/data/data/com.termux/files/usr/bin/bash
```bash
cp ~/.myfuncs.sh ~/
cp ~/.bashrc ~/
cp ~/.zshrc ~/
cp ~/.profile ~/

source ~/.bashrc

echo "✅ Termux 配置已恢复"
EOF
```
# ================================
# 5. 赋予脚本执行权限
# ================================
```bash
chmod +x config/termux/restore.sh
```
# ================================
# 6. 提交并推送到 GitHub（永久保存）
# ================================
```bash
git add config/termux
git commit -m "备份 Termux 配置 + restore.sh"
git push gitdemo master
```
🟢 第二部分：新手机 / 重装 Termux（以后用）
# ================================
# 1. 切换源
# ================================
```bash
termux-change-repo
```
# ================================
# 2. 读取本地权限
# ================================
```bash
termux-setup-storage
```
# ================================
# 3. 安装基础环境
# ================================
```bash
pkg update -y
pkg install git zsh -y
```
# ================================
# 4. 安装主题
# ================================
```bash
. <(curl -L gitee.com/mo2/zsh/raw/2/2)
```
# ================================
# 5. 克隆仓库（包含配置）
#    ⚠️ 第一次会要求输入：
#    - GitHub 用户名
#    - Personal Access Token（不是密码）
# ================================
```bash
git clone https://github.com/sxgpyjg/sxgpyjg.github.io.git gitdemo
```
# ================================
# 6. 一键恢复环境
# ================================
```bash
cd gitdemo
./config/termux/restore.sh
```
# ================================
# 7. 以后日常写作
# ================================
```bash
yjg "我的新文章"
```
