一键打包
```js
tar -czvf termux_full_backup.tar.gz \
  ~/gitdemo \
  ~/.bashrc \
  ~/.myfuncs.sh
```
----------备份↑----恢复↓----------

确认包在新终端里
```js
ls ~ | grep termux_full_backup.tar.gz
```
如果没有拷进来
```js
cp ~/storage/shared/termux_full_backup.tar.gz ~/
```
绝对路径模式解包
```js
cd /
tar -xzvf ~/termux_full_backup.tar.gz
```
加载
```js
source ~/.bashrc
```
验证
```JS
ls ~/gitdemo/_posts
git -C ~/gitdemo status
```
